```
Browser SPA (Scrutari)
  ├── Heuristic Engine (36+26 signals, 95 maxWeight)
  ├── ML Model (ONNX Runtime Web, ~9KB)
  └── Anomaly Detector (z-score, planned)
           │
           ▼
Server (Netlify)
  ├── /api/submit      — Deduplicated fingerprint storage
  ├── /api/analysis    — Entropy, confusion matrix, baselines
  ├── /api/classify    — GeoIP (replaces ipinfo.io)

  ├── /api/og          — Dynamic OG score card images
  ├── /api/model/*     — ML model metadata
  └── /api/health + 40+ honeypot paths — Bot traps
```

## Honeypot/Tarpit

- 40+ paths (admin, login, .env, dashboard, team, billing, staging, API, etc.)
- Content drift (visit-dependent numbers, dates, team members)
- Diurnal cycles (team status matches time of day)
- Visit tracking via `__hp_visit` cookie
- LLM prompt injection (hidden HTML comments + data-attributes)
- GraphQL introspection trap (`/api/graphql` returns fake schema)
- WordPress probe traps (`/xmlrpc.php`, `/wp-json/`)
- Stealth JS tracking (canvas, webdriver, automation globals — bot-only via sendBeacon)
- GDPR-compliant: tracking only fires for known bots/crawlers
